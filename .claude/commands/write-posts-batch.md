# 원소 포스트 배치 작성 에이전트

원소 심볼 목록: $ARGUMENTS

(쉼표 또는 공백으로 구분, 예: `Fe,Cu,Ag` 또는 `Fe Cu Ag`)

## 목적
여러 원소에 대해 각각 **흥미롭고 창의적인 주제**로 3개씩 양질의 블로그 포스트를 작성합니다.

---

## 주제 선정 가이드

### 8가지 주제 카테고리 (각 원소마다 다른 3개 선택)

1. **역사/발견 스토리**: 발견의 드라마, 과학자들의 경쟁, 이름의 유래
2. **과학적 특성의 비밀**: 독특한 물리/화학적 성질, 동소체, 결합 구조
3. **미래 기술/혁신**: 첨단 기술, 미래 에너지, 의료, 우주 탐사
4. **일상 속 숨은 주역**: 매일 만나지만 모르는 역할, 음식, 전자기기, 건강
5. **환경/지구/우주**: 기후변화, 별의 탄생, 우주에서의 역할
6. **문화/예술/신화**: 문화적 상징, 예술작품, 신화, 전설
7. **의외의 연결**: 전혀 관련 없어 보이는 분야와의 연결
8. **위험과 안전**: 원소의 독성, 위험성, 역사적 사고

## 제목 작성 패턴

### 좋은 제목 예시
- 질문형: "풍선은 왜 하늘로 날아갈까?"
- 대비/반전: "탄소: 지킬 앤 하이드"
- 숫자/구체성: "10초 만에 녹는 금속, 갈륨의 비밀"
- 드라마틱: "힌덴부르크 참사: 수소의 어두운 역사"
- 대결 구도: "수소차 vs 전기차: 승자는?"

### 피해야 할 제목
- "XX원소에 대하여" (지루함)
- "XX의 특징과 용도" (교과서 스타일)

## 콘텐츠 요구사항

### 각 포스트 구조
```
## 도입 (훅) - 흥미로운 질문/상황으로 시작
## 본론 1 - 핵심 내용
## 본론 2 - 추가 정보/다른 관점
## 본론 3 (선택) - 심화
## 마무리 - 인사이트, 여운
```

### 필수 요소
- **구체적 숫자**: "약 75%", "138억 년 전"
- **실제 예시**: 제품, 사건, 장소
- **비유 활용**: 어려운 개념을 친숙하게
- **스토리 흐름**: 단순 나열 X

### 분량
- 각 포스트: **1000-1800자**
- 소제목: 4-6개
- summary: 2-3문장
