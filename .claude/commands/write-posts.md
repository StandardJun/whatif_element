# 원소 블로그 포스트 작성 에이전트

원소 심볼: $ARGUMENTS

## 목적
주어진 원소에 대해 **흥미롭고 창의적인 주제**로 3개의 양질의 블로그 포스트를 작성합니다.

## 작업 순서

1. **원소 정보 수집**
   - `src/data/elements.ts`에서 해당 원소의 기본 정보 확인
   - `src/data/elementExtras.ts`에서 추가 정보 확인
   - 웹 검색으로 추가적인 흥미로운 사실 조사

2. **기존 포스트 스타일 참고** (필수!)
   - `src/data/posts.ts`의 H(수소), C(탄소) 포스트 분석
   - `src/data/posts/group1.ts`의 He(헬륨) 포스트 분석

3. **창의적인 주제 3개 선정 후 포스트 작성**

---

## 포스트 주제 선정 가이드

### 좋은 주제의 특징
- **스토리가 있다**: 단순 정보 나열 X, 이야기 흐름 O
- **의외성이 있다**: "이 원소가 이런 데 쓰인다고?"
- **일상과 연결된다**: 독자의 삶과 관련된 내용
- **질문을 던진다**: 호기심을 자극하는 화두

### 주제 카테고리 (각 원소마다 다른 3개 카테고리에서 선택)

1. **역사/발견 스토리**
   - 발견 당시의 드라마틱한 에피소드
   - 이름의 유래와 숨겨진 이야기
   - 과학자들의 경쟁과 실수
   - 예: "힌덴부르크 참사: 수소의 어두운 역사"

2. **과학적 특성의 비밀**
   - 독특한 물리/화학적 성질
   - 동소체, 결합 구조의 신비
   - 예: "탄소: 그의 두 번째 이름은 지킬 앤 하이드" (다이아몬드 vs 흑연)

3. **미래 기술/혁신**
   - 첨단 기술에서의 역할
   - 미래 에너지, 의료, 우주 탐사
   - 예: "수소차 vs 전기차: 미래 자동차의 승자는?"

4. **일상 속 숨은 주역**
   - 매일 만나지만 모르는 원소의 역할
   - 음식, 전자기기, 건강 등
   - 예: "스마트폰 안에 숨은 희토류의 비밀"

5. **환경/지구/우주**
   - 기후변화, 환경오염과의 관계
   - 우주에서의 역할, 별의 탄생
   - 예: "탄소발자국: 당신이 지구에 남기는 흔적"

6. **문화/예술/신화**
   - 원소와 관련된 문화적 상징
   - 예술작품, 신화, 전설
   - 예: "금: 왜 인류는 수천 년간 이 금속에 집착했을까?"

7. **의외의 연결**
   - 전혀 관련 없어 보이는 분야와의 연결
   - 예: "헬륨 목소리의 과학: 왜 목소리가 변할까?"

8. **위험과 안전**
   - 원소의 독성, 위험성
   - 안전하게 다루는 방법
   - 예: "수은: 왜 체온계에서 사라졌을까?"

---

## 제목 작성 가이드

### 좋은 제목 패턴
- **질문형**: "풍선은 왜 하늘로 날아갈까?"
- **대비/반전**: "탄소: 지킬 앤 하이드"
- **숫자/구체성**: "10초 만에 녹는 금속, 갈륨의 비밀"
- **드라마틱**: "힌덴부르크 참사: 수소의 어두운 역사"
- **미래 제시**: "수소차 vs 전기차: 승자는?"
- **일상 연결**: "스마트폰 배터리가 불타는 이유"

### 피해야 할 제목
- "XX원소에 대하여" (지루함)
- "XX의 특징과 용도" (교과서 스타일)
- 단순 정보 나열형

## 콘텐츠 작성 가이드

### 구조 (각 포스트)
```
## 도입 (훅)
- 흥미로운 질문이나 상황으로 시작
- 독자의 호기심 유발
- 2-3문단

## 본론 1
- 핵심 내용 전개
- 구체적인 예시, 숫자, 비유 활용

## 본론 2
- 추가 정보 또는 다른 관점
- 리스트, 비교표 등 활용 가능

## 본론 3 (선택)
- 심화 내용 또는 관련 주제

## 마무리
- 인사이트 또는 여운 남기기
- 다음 글 유도 또는 생각할 거리
```

### 문체
- **대화체**: "~알고 계셨나요?", "~해봤나요?"
- **비유 활용**: 어려운 개념을 친숙한 것에 비유
- **감탄사 적절히**: "놀랍게도", "사실은"
- **이모지 최소화**: 강조점에만 1-2개

### 필수 포함 요소
- **구체적인 숫자**: "약 75%", "138억 년 전", "5분이면"
- **실제 예시**: 실제 제품, 사건, 장소
- **과학적 정확성**: 틀린 정보 없이
- **독자 참여**: 질문, 생각해볼 거리

### 분량
- 각 포스트: **1000-1800자** (충분한 깊이)
- 소제목: 4-6개
- summary: 2-3문장 (클릭 유도)

## 참고할 기존 포스트 예시

### 수소 - 발견/우주 스토리
```
title: '수소: 우주의 시작, 빅뱅의 첫 번째 자녀'
→ 빅뱅에서 수소의 탄생, 별의 연료, 물의 구성, 미래 에너지
```

### 수소 - 미래 기술
```
title: '수소차 vs 전기차: 미래 자동차의 승자는?'
→ 전기차 시대에 왜 수소차?, 장단점 비교, 공존의 미래
```

### 수소 - 역사/사건
```
title: '힌덴부르크 참사: 수소의 어두운 역사'
→ 1937년 비극, 왜 수소였나, 현대의 안전 기술
```

### 탄소 - 과학적 특성
```
title: '탄소: 그의 두 번째 이름은 지킬 앤 하이드'
→ 다이아몬드 vs 흑연, 구조의 비밀, 그래핀, 생명의 근간
```

### 헬륨 - 일상/과학
```
title: '헬륨 목소리의 과학: 왜 목소리가 변할까?'
→ 소리의 전달, 헬륨에서 속도, 반대로 낮아지는 SF6, 안전 주의
```
